When ever user uploads an image in music app, where it should store and how http should pick from there, explains below,

In settings.py file of the Django project,
Go to end of the file and add two variables as follows,
Add below lines,
#This is the file path directory where we are putting all of the files loaded
MEDIA_ROOT = os.path.join(BASE_DIR, 'media')

#Browser use for accessing the files over http
MEDIA_URL = '/media/'

Add website urls as shown in file urls.py in this project.
from django.conf import settings
from django.conf.urls.static import static
